### P2-2 설계안 검증 ⬜

#### 검증 결과

**목표 달성 여부:**
- 모의투자 관련 코드 전부 제거 → ✅ 설계 완전 달성 가능

**설계 보완 사항 (P2-1에서 놓친 것):**
- `pnl_chart.py` line 297 에 `_is_paper()` 호출이 1개 더 있었음 → Phase E 에 추가 반영

**위험 요소 검토:**

| 위험 | 내용 | 대응 |
|------|------|------|
| 실 주문 발생 | 삭제 완료 후 주문이 실제 체결됨 | 의도된 동작 (실투자 목적) |
| API 키 없을 때 | login() 실패 → run_all.py 가 sys.exit(1) | ✅ 안전하게 중단됨 |
| _check_login() | `not _paper_trading` 조건 제거 시 로직 변화 없음 | ✅ 실계좌에서는 동일 동작 |
| `_paper_trading` global 제거 | login() 내 `global _upbit, _paper_trading` 수정 필요 | ✅ Phase A 에 포함 |
